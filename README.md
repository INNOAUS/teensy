### 提示

由于使用中文输入法，模拟输入错误命令；如果遇到中文输入法问题请加上两行代码

Keyboard.press(KEY_CAPS_LOCK); //按下大写键 这里我们最好这样写 不然大多数电脑在中文输入的情况下就会出现问题

Keyboard.release(KEY_CAPS_LOCK); //释放大写键
